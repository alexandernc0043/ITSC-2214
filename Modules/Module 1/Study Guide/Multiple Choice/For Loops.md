## [For Each Loop](https://www.w3schools.com/java/java_foreach_loop.asp)
```java
  for (type varName : arrayName) {
   Code to be executed
  }
```
```java
for(int i = 0; i < array.length; i++){
    Code to be executed
}
```
```java
for(int i : array){ 
  Code to be executed
}
```
```java
for(String s : array){ 
  Code to be executed
}
```
